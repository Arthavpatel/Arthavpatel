### 🏎️ Featured Project: [Monte Carlo F1 Race Strategy Simulator](https://github.com/Arthavpatel/monte-carlo-f1-strategy)

A Monte Carlo simulation that ranks two-stop tyre strategies by combining real FastF1 telemetry (tyre degradation, fuel burn, pit-stop loss) with stochastic race events (Safety Car, VSC, red flag) derived from historical track status. Runs 10,000 simulated races per strategy to estimate the expected race time *and* its distribution — the way a real strategy group reasons about pit calls.
